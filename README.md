# P2PForYourThings
C++ APIs for you to easily setup and communicate via smart P2P (using available p2p modules among Wi-Fi Direct / Bluetooth / Bluetooth LE)

= Supported Devices =
Raspberry-pi 3
NVIDIA Jetson-TX1
